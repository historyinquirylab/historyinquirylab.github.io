Подключение

Введите токен GitHub с правом записи в репозиторий. Токен хранится только в этом браузере.

Как получить токен
  1. Откройте Fine-grained tokens → Generate new.
  2. Resource owner: historyinquirylab. Repository access: Only select repositories → historyinquirylab.github.io.
  3. Permissions → Repository → Contents: Read and write.
  4. Сгенерируйте, скопируйте токен, вставьте выше и нажмите «Сохранить и подключиться».
